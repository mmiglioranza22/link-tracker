## Description

Link tracker created with [Nest](https://github.com/nestjs/nest).

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

Postman API ([access the shared workspace](https://www.postman.com/orbital-module-astronomer-66959558/workspace/link-tracker/overview))
